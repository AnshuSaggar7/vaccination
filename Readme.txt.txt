Run the pk_dash file in jupyter. It will take around 660 seconds to run the first line of code as code for prediction is imported which is pk_vaccination.ipynb.

The pagekite is just another python program which we found (https://pagekite.net/home/) and (https://www.sitepoint.com/accessing-localhost-from-anywhere/)
This program makes our local server public and can help to share our final dashboard by typing the url which is set upon registering in pagekite website.

the following command was used in terminal to convert my local host on port 8050 to go public using anshu.pagekite.me domain which was set up during registeration on the pagekite website.

pagekite.py 8050 anshu.pagekite.me
